### 四、使用教程

1. ​​上传图片​​：点击左侧“原图”和右侧“掩码图”按钮导入本地文件。

2. ​​选择模式​​：切换顶部按钮（滑动/淡入淡出/分屏）调整对比方式。

3. ​​细节分析​​：使用缩放控件放大关键区域，滑动条调节透明度。
